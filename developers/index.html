GardenNet Login Library

GardenNet Login Library is a plugin for other websites which we want them to login with a GardenNet account but however user registration only works in our GardenNet website. You can download the libaray here or scroll down to the Downloads. When you finish the download open the folder, then open "index.html"(make sure your internet connection is stable), then you will see the GardenNet LogIn box. If you open "index.html" in a code editor you will see a few firebase libraries were attached underneath the body element. The firebase libraries are very important because if you don't attach it in your website the GardenNetLoginLibrary will not work. The library is opensource so you can remax or design whatever you want.

The firebase liberies that you need to attach in your "index.html"

First you need to declaire a constant variable which contain GardenNet class.
const GN = new GardenNet;
To initialize the login data you need to type this code at your script file.
GN.init();
To pop up the login Box you need to type this code.
GN.login();
If the user successfully login you need to get his user data but don't worry the GardenNet Login Library has an object call userdata you can get the UserName like this console.log(userdata.username); and you can also log all the userdata like this console.log(userdata); so you can see all of userdata object in browser console.

Explorer Usages

You can go to the Explorer just by clicking here however it can't work because you didn't fill in the gid and gname parameters. "gid" parameter was used to set the UserID of the author. "gname" parameter was used to set the UserID of the author. "gname" parameter was used to set the Title of the post that was created in the author's post box.
